Windows 11：新功能一览

Published  2023/08/05    Updated  2024/02/27

一、全新开始菜单和任务栏

  1. 开始菜单

开始菜单是 Windows 11 中的重大变化之一。它更加简洁、现代，并且具有更多功能。新的开始菜单位于屏幕的中央，并且可以根据你的喜好进行自定义。你可以添加或删除应用程序、更改壁纸，甚至可以更改菜单的颜色和透明度。

要自定义开始菜单，只需右键单击开始按钮，然后选择 “个性化”。在 “个性化” 设置中，你可以找到各种选项来更改开始菜单的外观和功能。

  1. 任务栏

任务栏也是 Windows 11 中的新增功能之一。它位于屏幕的底部，并且包含各种应用程序图标和按钮。任务栏也可以根据你的喜好进行自定义。你可以添加或删除应用程序图标、更改任务栏的颜色和透明度，甚至可以更改任务栏的位置。

要自定义任务栏，只需右键单击任务栏，然后选择 “个性化”。在 “个性化” 设置中，你可以找到各种选项来更改任务栏的外观和功能。

  1. 小部件

小部件是 Windows 11 中的另一项新增功能。小部件位于屏幕的右侧，并且可以让你快速访问常用信息，如天气、新闻、股票和体育赛事。小部件也可以根据你的喜好进行自定义。你可以添加或删除小部件、更改小部件的颜色和透明度，甚至可以更改小部件的位置。

要自定义小部件，只需右键单击小部件面板，然后选择 “个性化”。在 “个性化” 设置中，你可以找到各种选项来更改小部件的外观和功能。

二、全新多任务处理功能

  1. 任务视图

任务视图是 Windows 11 中的多任务处理功能之一。它可以让你查看所有打开的窗口的缩略图，并且快速切换到所需的窗口。要打开任务视图，只需按键盘上的 “Win+Tab” 组合键即可。

  1. 虚拟桌面

虚拟桌面是 Windows 11 中的另一项多任务处理功能。它可以让你创建多个虚拟桌面，并分别在每个桌面上运行不同的应用程序。要创建虚拟桌面，只需点击任务栏上的 “任务视图” 按钮，然后点击 “新建桌面” 按钮即可。

  1. 分组窗口

分组窗口是 Windows 11 中的另一项多任务处理功能。它可以让你将多个窗口分组在一起，并同时查看这些窗口。要分组窗口，只需将一个窗口拖放到另一个窗口上即可。

三、全新游戏体验

  1. DirectStorage

DirectStorage 是 Windows 11 中的全新游戏功能。它可以减少游戏加载时间，并提高游戏性能。DirectStorage 通过直接将游戏数据从存储设备加载到显卡，而无需经过中央处理器，从而减少了游戏加载时间。

  1. Auto HDR

Auto HDR 是 Windows 11 中的另一项全新游戏功能。它可以自动将标准动态范围 (SDR) 游戏转换为高动态范围 (HDR) 游戏，从而提高游戏画质。Auto HDR 通过使用算法来分析游戏的颜色和亮度，并自动生成 HDR 图像。

  1. Xbox Game Pass

Xbox Game Pass 是 Windows 11 中的订阅服务。它可以让你访问数百款游戏，包括最新发布的游戏、经典游戏和独立游戏。Xbox Game Pass 的月费为 9.99 美元，并且可以在 Xbox、PC 和移动设备上使用。

四、全新安全功能

  1. Windows Security

Windows Security 是 Windows 11 中的安全中心。它可以保护你的电脑免受病毒、恶意软件、间谍软件和其他威胁。Windows Security 包括各种安全功能，如防病毒扫描、实时保护、防火墙和家长控制。

  1. Windows Hello

Windows Hello 是 Windows 11 中的生物识别身份验证系统。它可以让你使用指纹、面部识别或 PIN 码来登录你的电脑。Windows Hello 比传统的密码更加安全，并且更加方便。

  1. Device Encryption

Device Encryption 是 Windows 11 中的加密功能。它可以加密你的电脑上的数据，以防止未经授权的人访问。Device Encryption 可以保护你的个人文件、照片、视频、音乐和其他数据。

五、全新辅助功能

  1. Narrator

Narrator 是 Windows 11 中的屏幕阅读器。它可以朗读屏幕上的文本、按钮和链接，从而帮助视障人士使用电脑。Narrator 可以与各种屏幕阅读器兼容，如 JAWS 和 NVDA。

  1. Magnifier

Magnifier 是 Windows 11 中的放大镜。它可以放大屏幕上的文本、按钮和链接，从而帮助视障人士看清屏幕上的内容。Magnifier 可以与各种放大镜兼容，如 ZoomText 和 SuperNova。
